# Metadata

This folder contains business-level metadata and contextual information.

## Purpose
Store high-level business information including:
- Legal business information
- Compliance and regulatory framework
- Service area overview
- Data governance policies
- Business context for LLM understanding

## File Format
- **JSON files**: Structured metadata conforming to `/schema/metadata-schema.json`
- **Markdown files**: Human-readable business information

## Data Policy
- ✅ Include only verified business information
- ✅ Accurate compliance and regulatory data
- ✅ Clear data governance policies
- ❌ No promotional content
- ❌ No unverified claims
- ❌ No invented history or achievements

## Key Files
- `business-info.json` - Core business information
- `compliance.json` - Regulatory and compliance data
- `service-area.json` - Geographic service area information
- `data-governance.json` - Data accuracy and verification policies

## Template
See `_template.json` for the structure to use when adding metadata.
