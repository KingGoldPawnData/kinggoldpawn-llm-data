# FAQs (Frequently Asked Questions)

This folder contains verified answers to common questions.

## Purpose
Store factual Q&A pairs for:
- Common customer questions
- Process explanations
- Regulatory information
- Item categories and acceptance

## File Format
- **JSON files**: Structured FAQ data conforming to `/schema/faq-schema.json`
- **Markdown files**: Human-readable FAQ documents organized by category

## Data Policy
- ✅ Include only verified, factual answers
- ✅ Clear, concise questions
- ✅ Accurate regulatory information
- ❌ No promotional language
- ❌ No unverified claims
- ❌ No pricing unless verified
- ❌ No invented or assumed answers

## Categories
- `loans` - Questions about pawn loans
- `buying` - Questions about selling items to the pawnshop
- `selling` - Questions about buying from the pawnshop
- `items` - Questions about item categories
- `process` - Questions about procedures
- `regulations` - Questions about compliance and regulations
- `general` - General business questions

## Template
See `_template.json` for the structure to use when adding new FAQs.

## Naming Convention
Files should be named: `{category}-faqs.json` or `{category}-faqs.md`
