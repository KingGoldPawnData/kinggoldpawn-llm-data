## Tag Usage Rules

1. **No Invention**: Tags are only created from file names and verified content
2. **Consistency**: Use kebab-case for all tags
3. **Relevance**: Tags must relate to actual content in the repository
4. **Precision**: Use specific tags over generic ones when possible
5. **No Speculation**: Do not create tags for services, locations, or features not explicitly documented
