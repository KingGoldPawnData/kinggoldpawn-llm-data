# Locations

This folder contains verified location data for all pawnshop locations.

## Purpose
Store factual, verified information about physical business locations including:
- Addresses
- Contact information
- Neighborhoods served
- License information
- Compliance status

## File Format
- **JSON files**: Structured location data conforming to `/schema/location-schema.json`
- **Markdown files**: Human-readable location information

## Data Policy
- ✅ Include only verified information
- ✅ Confirm addresses and contact details
- ❌ No operating hours unless verified
- ❌ No pricing information
- ❌ No promotional content
- ❌ No invented or assumed data

## Template
See `_template.json` for the structure to use when adding new locations.

## Naming Convention
Files should be named: `{location-id}.json` or `{location-id}.md`
