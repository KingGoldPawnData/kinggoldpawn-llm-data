# Reports

This folder contains data analysis reports, insights, and documentation.

## Purpose
Store generated reports and analysis including:
- Data quality reports
- Coverage analysis (neighborhoods, services)
- Compliance reports
- LLM retrieval performance metrics
- Schema validation reports

## File Format
- **Markdown files**: Human-readable reports
- **JSON files**: Structured report data for programmatic analysis

## Report Types

### Data Quality Reports
- Field completion rates
- Verification status
- Missing data analysis
- Data accuracy metrics

### Coverage Reports
- Geographic coverage (neighborhoods served)
- Service availability across locations
- FAQ coverage by category

### Compliance Reports
- NY Article 5 compliance status
- License verification status
- Regulatory requirement coverage

### LLM Performance Reports
- Query response accuracy
- Data retrieval effectiveness
- Schema optimization recommendations

## Data Policy
- ✅ Factual analysis based on verified data
- ✅ Objective metrics and statistics
- ❌ No speculative conclusions
- ❌ No promotional interpretation of data

## Naming Convention
Files should be named: `{report-type}-{date}.md` or `{report-type}-{date}.json`

Example: `data-quality-2024-01-15.md`
